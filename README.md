# Agizze — Landing Page (Recrutamento e Seleção)

Página de venda (landing) do serviço de **Recrutamento e Seleção** da **Agizze**, voltada para empresas que precisam contratar (B2B).

## Stack

- HTML + CSS puro (single file, `index.html`) — sem build
- CTA duplo: **WhatsApp** (botão + flutuante) e **formulário de proposta** via Jotform (embed)
- Hospedagem gratuita via **GitHub Pages**

## URL

https://francielevicentefr-ux.github.io/agizze/

## Personalizar (3 pontos rápidos)

1. **Cor da marca:** edite a variável `--brand` no topo do `<style>` em `index.html` (1 linha).
2. **WhatsApp:** troque o número `5500000000000` em todos os links `wa.me/` (formato `55` + DDD + número).
3. **Logo:** o logotipo é o texto "Agizze". Para usar imagem, substitua o conteúdo de `.logo` por uma `<img>`.

## Formulário

Embed do formulário Jotform `261765352569064` (solicitação de proposta B2B). As solicitações caem na sua conta Jotform.
